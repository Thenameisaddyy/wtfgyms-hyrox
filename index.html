What is HYROX?
The world's fastest growing
indoor fitness race.

HYROX is a global fitness race held indoors — same format in every city worldwide. Run 1km, complete a workout challenge, repeat 8 times. 8km of running + 8 physical challenges. Open to everyone.

Same format in 30+ countries
🏃
Run + Challenge × 8
Every round is 1km of running followed by one physical challenge. Eight rounds total — 8km running + 8 challenges.
